Algoritmos e Programao Estruturada

APE-U1S1

  • Algoritmo uma sequncia ordenada de passos, que devem ser seguidos, para atingir um objetivo.
  • Simplificadamente, mostra como se resolve o problema.
  • No caso de programao, mostra o que deve ser feito para solucionar o problema.
  • O algoritmo vem antes da escrita de um programa (implementao).
  • A partir do algoritmo pode-se escrever vrios programas em linguagens diferentes (implementaes), para resolver o problema descrito.
  • Pode ser divido em 3 partes: Entrada, Processamento, Sada.
  • No existe um padro bem definido para escrever algoritmos. Pode-se utilizar:
    • Diagrama de bloco.
    • Linguagem natural.
    • Pseudocdigo que lembra uma linguagem de programao simplificada.
    • Prticas comuns utilizadas em artigos e livros.
    • Python simplificado omitindo os detalhes.
    • Alguma outra prtica comum existente.

Linguagem natural

  • o que as pessoas conversam no dia-a-dia. Ex.:
  1. Incio.
  2. Entrar com o primeiro valor (nota do primeiro bimestre).
  3. Entrar com o segundo valor (nota do segundo bimestre).
  4. Realizar a soma do primeiro valor com o segundo.
  5. Realizar a diviso da soma dos valores por dois (mdia das notas dos bimestres).
  6. Armazenar o valor encontrado.
  7. Mostrar na tela o resultado da mdia.
  8. Se a mdia do aluno for maior ou igual a seis.
  9. O aluno ser considerado aprovado.
  10. Seno est reprovado.
  11. Fim.

Diagrama de blocos (fluxograma)

Smbolo Significado Descrio
Terminal Representa o incio ou o fim de um fluxo lgico. Em alguns casos define as sub-rotinas
Entrada manual Determina a entrada manual dos dados, geralmente por meio de um teclado
Processamento Representa a execuo de aes de processamento
Exibio Mostra o resultado de uma ao, geralmente atravs da tela de um computador
Deciso Representa os desvios condicionais nas operaes de tomada de deciso e laos condicionais para repetio de alguns trechos do programa
Preparao Representa a execuo de um lao incondicional que permite a modificao de instrues do lao
Processo predefinido Define um grupo de operaes relacionadas a uma sub-rotina
Conector Representa pontos de conexes entre trechos de programas, que podem ser apontados para outras partes do diagrama de bloco
Linha Representa os vnculos existentes entre os smbolos de um diagrama de blocos

Paradigmas

Padro de solues de problemas, jeito de programar, recursos da linguagem. Algumas linguagens apresentam vrios paradigmas, ex.: Python (orientado a objetos, imperativa, funcional, e outros)

  • Programao imperativa
    • O programa um passo-a-passo de como o resultado deve ser obtido. Ex: C.
  • Programao orientada a objetos
    • O programa uma coleo de objetos que se inter-relacionam. Ex.: C++.
  • Programao funcional
    • Programao usando expresses e funes. Ex.: Lisp.
  • Programao lgica
    • O programa uma declarao dos resultados. Ex.: Prolog.

Sintaxe

  • So a regras e como o programa escrito
    • Comandos, expresses, variveis, etc.
  • geralmente escrito na sequncia que se espera que seja executado.

C - operadores aritmticos

In[53]:
%%writefile a.c

#include <stdio.h>

int main() {
    float a1, a2, b, c;

    a1 = 1 + 2 - 3 * 4 / 5;
    a2 = 1 + 2 - 3 * 4 / 5.0;
    b = 2 % 3;
    c = -2 + (-3);

    printf("%f, %f, %f, %f\n", a1, a2, b, c);
}

Overwriting a.c
In[54]:
! gcc a.c && ./a.out

1.000000, 0.600000, 2.000000, -5.000000
  • Apesar da varivel a1 ser float, os clculos so feitos usando os tipos dos nmeros da expresso, pela ordem de prioridade.
  • Em a1 o compilador inicia pela multiplicao e diviso, considera os nmeros como inteiros, e usa aritmtica inteira.
  • Em a2 o 5.0 faz com que o clculo use aritmtica de ponto flutuante.

C - printf()

printf (expresso de controle, listas de argumentos)

Ex.: printf("%d, %f, %s\n", a, b, c);

Cdigo Funo
%c escrita de apenas um caractere
%d nmeros inteiros decimais
%e nmeros em notao cientfica
%f nmeros reais (ponto flutuante)
%g %e ou %f no formato mais curto.
%o nmeros octais
%s srie de caracteres
%u nmero decimal sem sinal
%x nmero hexadecimal
%n entrada de uma sequncia de caracteres
\n pula uma linha
