Escopo e passagem de parmetros

Varivel local

In[3]:
#include<stdio.h>

void teste() {
    int x = 10;

    printf("\n Valor de x na funcao test() = %d", x);
}

int main() {
    int x = 20;

    teste();  //"procedimento" (no retorna valor)
    printf("\n Valor de x na funcao main() = %d", x);
}

 Valor de x na funcao test() = 10
 Valor de x na funcao main() = 20

Varivel global

In[5]:
#include<stdio.h>

int x = 10;

void teste() {

    x = 20;
    printf("%d, ", x);
}

int main() {
    printf("%d, ", x);
    teste();
    x = 30;
    printf("%d, ", x);
}

10, 20, 30,
  • Global permanece na memria ocupando espao durante todo o tempo de execuo.
    • A local libera o espao no final da funo.
  • Dificulta modularizao e reuso.
  • Uma mudana na varivel acaba se propagando para o programa todo, o programa pode funcionar assim mesmo e o bug fica difcil de encontrar.
  • Fica difcil de testar mdulos separadamente, ou isolar um trecho para testar.
    • Se mudar a varivel apenas naquele trecho, afeta o programa inteiro.
  • Se quiser alterar a varivel em um programa de 1000 linhas precisa percorrer todos os arquivos e toda estrutura de diretrios.
  • Pode levar a problemas de coliso ao unir vrios mdulos e bibliotecas.
  • Uma aplicao seria usar para definir constantes que no mudam no programa todo.
  • Se existir uma global e uma local com o mesmo nome fica difcil identificar.

Exemplo do uso de variveis globais

#include<stdio.h>

float t1, t2;

}

    printf("\n Digite as duas temperaturas: ");
    scanf("%f %f", & t1, & t2);

#include<stdio.h>

int x = 10;

int main() {
    int x = -1;
    int b;

    {
        extern int x;
        b = x;
    }

    printf("\n Valor de x = %d", x);
    printf("\n Valor de b (x global) = %d", b);
}

 Valor de x = -1
 Valor de b (x global) = 10
  • Os colchetes definem um novo escopo, assim a varivel x no colide.

Passagem de parmetros para funes

  • Passagem por valor
    • O valor da varivel copiado
      • As variveis originais no so alteradas
    • Variveis locais so criadas, contendo o valor
    • Quando termina a funo, as variveis locais so liberadas
  • Passagem por referncia
    • O endereo da varivel (ponteiro) copiado
In[10]:
#include<stdio.h>

int somar(int a, int b) {
    return a + b;
}

int main() {
    int result;

    result = somar(10, 15);
    printf("\n Resultado da soma = %d", result);
}

 Resultado da soma = 25

#include<stdio.h>

void testar(int n1, int n2) {
    n1 = -1;
    n2 = -2;
    printf("\n\n Valores dentro da funcao testar(): ");
    printf("\n n1 = %d e n2 = %d", n1, n2);
}

int main() {
    int n1 = 10;
    int n2 = 20;

    printf("\n\n Valores antes de chamar a funcao: ");
    printf("\n n1 = %d e n2 = %d", n1, n2);

    testar(n1, n2);  // "procedimento" (a funo no retorna valor)

    printf("\n\n Valores depois de chamar a funcao: ");
    printf("\n n1 = %d e n2 = %d", n1, n2);
    return 0;
}


 Valores antes de chamar a funcao: 
 n1 = 10 e n2 = 20

 Valores dentro da funcao testar(): 
 n1 = -1 e n2 = -2

 Valores depois de chamar a funcao: 
 n1 = 10 e n2 = 20

Passagem por referncia

  • Ao invs do valor da varivel, passado o endereo de memria (ponteiro) da varivel
  • Cuidado:
    • A varivel no copiada
    • No criada uma nova varivel
    • A varivel no se torna local
    • Se alterar acaba alterando a varivel original da funo que chamou

Ponteiros

  • Operao com ponteiros:
    • "*" pode significar 2 coisas:
      • Declarar a varivel como tipo ponteiro ("declarao")
      • Acessar o contedo da varivel apontada pelo ponteiro ("valor")
    • "&" significa "retorna o endereo de memria da varivel" ("endereo")
In[16]:
#include<stdio.h>

void testar(int * n1, int * n2) {
    * n1 = -1;  // * = "contedo apontado por"
    * n2 = -2;
    printf("\n\n Valores dentro da funcao testar(): ");
    printf("\n n1 = %d e n2 = %d", * n1, * n2);
}

int main() {
    int n1 = 10;
    int n2 = 20;
    printf("\n\n Valores antes de chamar a funcao: ");
    printf("\n n1 = %d e n2 = %d", n1, n2);

    testar( & n1, & n2);

    printf("\n\n Valores depois de chamar a funcao: ");
    printf("\n n1 = %d e n2 = %d", n1, n2);
}


 Valores antes de chamar a funcao: 
 n1 = 10 e n2 = 20

 Valores dentro da funcao testar(): 
 n1 = -1 e n2 = -2

 Valores depois de chamar a funcao: 
 n1 = -1 e n2 = -2

Passagem de vetor

  • sempre feita implicitamente por referncia
  • No necessrio usar "*" e "&"
In[18]:
#include<stdio.h>

void inserir(int a[]) {
    int i = 0;

    for (i = 0; i < 3; i++) {
        printf("Digite o valor %d: ", i);
        scanf("%d", & a[i]);
    }
}

void imprimir(int b[]) {
    int i = 0;

    for (i = 0; i < 3; i++) {
        printf("\n numeros[%d] = %d", i, 2 * b[i]);
    }
}

int main() {
    int numeros[3];

    printf("\n Preenchendo o vetor... \n ");

    inserir(numeros);

    printf("\n Dobro dos valores informados:");

    imprimir(numeros);
}

 Preenchendo o vetor... 
 Digite o valor 0: 
Digite o valor 1: 
Digite o valor 2: 
 Dobro dos valores informados:
 numeros[0] = 20
 numeros[1] = 40
 numeros[2] = 60
