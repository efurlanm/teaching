Introduo ao Pandas

Dentre as diversas bibliotecas disponveis no repositrio PyPI, pandas um pacote Python que fornece estruturas de dados projetadas para facilitar o trabalho com dados estruturados (tabelas) e de sries temporais.

Estruturas de dados:

  • Series
  • DataFrame

Series um como um vetor de dados unidimencional capaz de armazenar diferentes tipos de dados.

DataFrame conjunto de Series.

Possuem como caracterstica a indexao das linhas

  • Cada linha possui um rtulo (nome) que o identifica

    • Pode ser string, inteiro, decimal, ou data

requests - biblioteca HTTP para ler pginas web

In[64]:
dados = []
for noticia in lista_noticias:
    data = noticia.contents[0].text.strip() + ', 2017'
    comentario = noticia.contents[1].strip().replace("", '').replace("", '')
    explicacao = noticia.contents[2].text.strip().replace("(", '').replace(")", '')
    url = noticia.find('a')['href']
    dados.append((data, comentario, explicacao, url))
