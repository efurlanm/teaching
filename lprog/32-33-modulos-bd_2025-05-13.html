Banco de dados

Ao criar uma aplicao em uma linguagem de programao que precisa acessar um sistema gerenciador de banco de dados relacional (RDBMS), uma vez que so processos distintos, preciso criar uma conexo entre eles. Aps estabelecida a conexo, possvel (de alguma forma) enviar comandos SQL para efetuar as aes no banco (RAMAKRISHNAN; GEHRKE, 2003). Para fazer a conexo e permitir que uma linguagem de programao se comunique com um banco de dados com a utilizao da linguagem SQL, podemos usar as tecnologias ODBC (Open Database Connectivity) e JDBC (Java Database Connectivity).

Agora que temos uma conexo com um banco de dados, vamos utilizar uma instruo DDL da linguagem SQL para criar a tabela fornecedor. O comando SQL que cria a tabela fornecedor est no cdigo a seguir e foi guardado em uma varivel chamada ddl_create. Caso execute todas as clulas novamente, certifique-se de apagar a tabela no banco, para evitar o erro.
